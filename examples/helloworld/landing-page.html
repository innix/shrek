Hello world! You are running this web page on Tor!

This is a very simple web page, and if you're reading this then you've browsed to it from a Tor browser. The .onion address being used by Tor was generated by Shrek, a vanity .onion address generator written in Go.

Shrek is an open source project available on GitHub. It can be used as a CLI tool or as a library in your Go code.
